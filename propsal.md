# 企画書：Tiny Pixel Master
**〜 組み込みOLED専用・超軽量アニメーション制作エディタ 〜**

## 1. コンセプト
「マイコンのメモリ制限を、クリエイティビティの制約に変える。」
0.96インチOLED特有の表示特性を再現し、MicroPythonやArduinoでの実装に最適化したコードを出力する、8bit Painter風の操作感を持つWebベースのエディタ。

## 2. ターゲット
- マイコン（ESP32, Raspberry Pi Pico, Arduino等）で電子工作をする人
- OLEDに滑らかなアニメーションを表示させたいが、メモリ不足に悩んでいる人
- ドット絵をコード化する手間（配列変換）を自動化したい人

## 3. 主要機能（神機能）

### ① ハードウェア・シミュレート・プレビュー
- **リアルな質感:** OLED特有の発光（青色・白色）や、物理的なピクセル間の隙間を再現。
- **2色モデル対応:** 0.96インチ特有の「上部16pxが黄色、下が青色」の境界線を可視化。

### ② スライド式アニメーション管理 (PowerPointスタイル)
- **複数フレーム管理:** パワーポイントのように、スライド一覧でフレームを管理（複製、並び替え、削除）。
- **オニオンスキン:** 前後のフレームをうっすら透過表示し、動きの繋がりを確認。

### ③ デルタ圧縮出力（メモリ節約の要）
- **差分更新ロジック:** 前のフレームから「変化したドット」だけを抽出。
- **コード最適化:** 全画面を書き換えるのではなく、変化した範囲だけを更新するMicroPython/C++コードを自動生成し、RAM/Flash消費を劇的に削減。

### ④ 画像インポート & 2値化
- **PNG/JPG読み込み:** 任意の画像をキャンバスサイズにリサイズ。
- **高度な2値化:** 閾値（Threshold）調整に加え、誤差拡散法（ディザリング）により写真もモノクロで表現可能。

---

## 4. UI/UX 設計（8bit Painterスタイル）
- **メインキャンバス:** 画面中央に配置。ピンチズーム・パン対応の生Canvas。
- **ツールバー（フローティング）:** ペン、消しゴム、バケツ、スポイト、Undo/Redoをシンプルに配置。
- **下部スライドストリップ:** フレーム一覧を横並びに表示。
- **サイドパネル:** - ディスプレイサイズ設定（128x64, 128x32, 128x128等）
    - エクスポート設定（MicroPython, C++ / 形式：Hex, Bytearray）

---

## 5. 技術スタック
- **Frontend:** React (State管理 / UI)
- **Drawing Engine:** Vanilla HTML5 Canvas API (高速なピクセル操作)
- **Format:** PWA (Progressive Web App)
    - オフライン動作対応
    - スマホのホーム画面に追加可能
    - ブラウザから直接画像を「共有」してインポート
- **Data Persistence:** IndexedDB / localStorage (制作データの自動保存)

---

## 6. 出力仕様

### MicroPython (優先)
- `framebuf` モジュールでそのまま読み込める `bytearray` 形式。
- 差分のみを `pixel(x, y, color)` または `blit()` で更新する最適化スクリプト。

### Arduino / C++
- `PROGMEM` を使用したメモリ節約型の定数配列。
- SSD1306用ライブラリ（Adafruit等）に準拠したデータ構造。

---

## 7. ロードマップ

### Phase 1: MVP（最小機能版）
- 128x64のCanvas描画機能。
- 単一フレームのMicroPython bytearray書き出し。
- 8bit Painter風の基本UI実装。

### Phase 2: アニメーション & プレビュー
- フレーム追加・複製機能。
- オニオンスキン実装。
- OLED質感シミュレーター。

### Phase 3: 最適化 & 連携
- 差分圧縮アルゴリズムの実装。
- 画像インポート・ディザリング機能。
- Web Serial APIによる実機への直接転送（デバッグ機能）。

---

## 8. デザインの方向性
- **テーマ:** ダークモード（OLEDの黒を基調）。
- **アクセントカラー:** ネオンブルー（OLEDの発光色）。
- **操作性:** モバイル・デスクトップ両対応のレスポンシブ設計。
